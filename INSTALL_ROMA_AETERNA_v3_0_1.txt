ROMA AETERNA v3.0.1 — УСТАНОВКА ФИКСА

Безопасный вариант:
1. Скопируйте папку игры целиком в резервное место.
2. Сохраните отдельно roma_savegame.json и roma_savegame.json.bak.
3. Замените roma_aeterna.py и roma_army_groups.py.
4. Добавьте новый файл roma_navy.py рядом с roma_aeterna.py.
5. Запустите: python roma_aeterna.py --self-test
6. Затем запустите игру обычным способом.

roma_economy.py, roma_ai_civilization.py и roma_nations.py менять не требуется.
Старое сохранение будет мигрировано автоматически.
