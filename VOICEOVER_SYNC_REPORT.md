# Синхронизация дерева технологий с готовой озвучкой

## Применённые подтверждённые решения

- `state_mints`: полный отрывок из Евангелия от Матфея о кесаревой монете.
- `customs_posts`: отдельный отрывок из Евангелия от Луки о мытарях.
- `fleet_bases`: цитата о Порте Юлия из Страбона.
- `provincial_archives`: новая цитата Плиния Младшего об архивных документах; повтор Светония удалён.
- `monument_commissions`: выбранная цитата из «Деяний Божественного Августа».
- Для технологий 83–100 источники записаны словами: «книга», «глава», «параграф», «письмо», «стихи» или «раздел», где это применимо.

## Что намеренно не изменено

Технологии 98–100 (`imperial_couriers`, `grain_contracts`, `ballista_corps`) сохранены в исходном виде. Два предложенных набора замен были отвергнуты, а окончательный новый набор в переписке не был утверждён. Их самовольная замена могла бы рассинхронизировать уже готовые WAV-файлы.

## Проверки

- Ровно 100 технологий и 100 цитат.
- Все ID цитат совпадают с ID дерева.
- Названия технологий в дереве и файле цитат совпадают.
- JSON успешно разбирается.
- Предпосылки дерева не содержат неизвестных ID.
